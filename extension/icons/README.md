# XPostAgent アイコン

## 必要なファイル

以下のサイズのPNGアイコンが必要です：

- `icon16.png` - 16x16px（ツールバー用）
- `icon48.png` - 48x48px（拡張機能管理画面用）
- `icon128.png` - 128x128px（Chrome Web Store用）

## アイコン作成手順

1. 128x128pxでアイコンをデザイン
2. 48x48px、16x16pxにリサイズ
3. このフォルダに保存

## 推奨デザイン

- Xのブランドカラー（黒背景、白ロゴ）または独自デザイン
- シンプルで視認性の高いデザイン
- 背景は透明または単色

## 仮のアイコン生成（開発用）

開発時は以下のサイトで仮アイコンを生成できます：
- https://favicon.io/
- https://realfavicongenerator.net/
